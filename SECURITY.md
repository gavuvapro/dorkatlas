# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Latest stable   |

## Reporting a Vulnerability

Please email **security@dorkatlas.dev** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and coordinate disclosure responsibly.

Thank you for helping keep DorkAtlas secure.
